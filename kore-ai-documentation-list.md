## Platform Overview

### What is Kore.ai?

**Kore.ai** is an enterprise-grade multi-agent orchestration platform that enables organizations to develop, deploy, and manage AI-powered business applications. It's a no-code/low-code platform for building AI agents that can interact with each other and handle specific tasks with precision.

**Official Website**: https://www.kore.ai

### Platform Value Proposition

- **No-Code Development**: Build intelligent virtual assistants without writing backend code
- **Enterprise-Ready**: Includes security, scalability, and governance features
- **Multi-Channel Support**: Deploy across 40+ channels (voice, chat, web, mobile)
- **AI-Powered**: Leverages advanced NLP, GenAI, and agentic AI capabilities
- **Cost Efficiency**: Reduces over 80% of common human-assisted interactions
- **Quick ROI**: Customers report $3.5M+ in cost efficiencies in Year 1

### Key Platform Components

**Two Main Paradigms:**

1. **Dialog Agents** (Orchestrated Autonomy)
   - Structured flow approach powered by DialogGPT
   - Enables dynamic conversational experiences
   - Ideal for defined business processes

2. **AI Agents** (Full Autonomy)
   - Reasoning-driven approach with human-like interaction
   - Multi-agent systems with persistent memory
   - Highest level of AI sophistication
